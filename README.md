# Sales Performance Dashboard (Power BI)

## Objective
To analyze sales performance by month, region, and category using Power BI.

## Tools Used
- Power BI
- Superstore Sales Dataset

## Visualizations
- Monthly Sales Trend (Line Chart)
- Sales by Region (Bar Chart)
- Sales by Category (Donut Chart)
- Region Filter (Slicer)

## Key Insights
- West region recorded the highest sales.
- Monthly sales show an overall upward trend.
- Technology category contributed the most to total sales.
